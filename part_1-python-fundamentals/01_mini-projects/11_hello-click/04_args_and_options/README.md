# Arguments and options

## Usage notes

```bash
$ python helloargs.py --help
Usage: helloargs.py [OPTIONS] NAME

Options:
  --count INTEGER  number of greetings
  --help           Show this message and exit.

$ python helloargs.py adri
Hello to adri

$ python helloargs.py adri --count 3
Hello to adri
Hello to adri
Hello to adri
```
